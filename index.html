Visual Studio Code

Visual Studio Code is a powerful text-editor and development environment for working on Code and developing websites. It includes many built in tools for making your life easier, such as a built-in file browser, suggestions for HTML tags, and syntax highlighting.

If you need an additional review on the Editor and its capabilities, please see the Visual Studio Code Demo (Password: interaction).

Visual Studio Code also includes the ability to install Extensions to enable additional features, in class we reviewed the following:

  • Prettier - Automatically format your code
  • GitLens - See History and Current Changes to Codebase
  • Live Server - Run a Local Server from the Project Folder

Local Development Environment / Chrome Developer Tools

When working locally, it can be helpful to debug a website in the browser. The best tools for doing so are the Chrome Developer Tools.. These tools are included in the browser and let you inspect individual HTML elements, and look at and temporarily modify their CSS in the 'Inspect' window pane. You can use 'Console' to debug and test Javascript, and finally use the 'Network' tab to check for which other files and requests are being made to gather all the content and files for current web page.

For additional information and a walk-through, please see the demonstration of Chrome Developer Tools (password: interaction).

Git & Github

Git is a piece of software that runs on your computer which allows you to manage "Versions" of your code and files. Whenever you make changes on anything in a Git folder, it will track those differences.

When you are done making changes, you make a "Commit" of those changes with a description so that you can see, and in the future review, the history and status of the project.

Github is a Microsoft-owned Company which allows sharing your code and projects online using Git. When you have "Pushed" your project online, you can then see its current history and state much like you would be able to locally, and now you can also share the code or use it for hosting a Github Page.

For additional information, please see the demonstration of Git & Github (password: interaction).

Github Pages

Github Pages are a free service offered by Github that allows you to host your website.

Much like we can use the Visual Studio Code extension "Live Server" to privately host a website on our local network, Github Pages allows you to deliver that website publicly to the internet.

For additional reference, please review the demonstration of Github Pages
