Core 2: Interaction Lab

Workshop #4 - Lore

Our in-class Workshop will be to take a pre-defined data set of items from Elden Ring and manipulate the data, and display it on the page in different forms.

Dataset: data.js

Please display the data on the page in at least three different ways of presenting the data using the Map, Sort, and Filter methods on the arrays. Some ideas include:

  • Use Map to show a differently formatted version of the Attack value.
  • Use Sort to display the weapons alphabetically, or by Attack value.
  • Use Filter to only display weapons with an attack value above a certain threshold.

If you have additional time, add some styling to make the page responsive and usable across mobile, tablet and desktop devices. You could also explore adding buttons to the page to let the user control which version is displayed on the page.
