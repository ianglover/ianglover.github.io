Workshop #1 - Typography on the Web

Styling the Homepage

Our in-class Workshop is to add a custom fonts and additional styling to your personal homepage.

Find a typeface that you like ( feel free to use the links and resources here or use your own ). Be sure to take notice of font licenses and only use fonts which are licensed for use in non-profit scenarios or for educational purposes.

Please use or demonstrate at least the following:

  • Create and import an external CSS file
  • Header Elements ( H1, ... )
  • Paragraph Elements ( P )
  • List Elements ( UL, OL, LI )
  • At least one CSS Class used on multiple HTML Elements
  • At least one styled ID
  • At least one non-default color
  • At least one Custom Font
