Core 2: Interaction Lab

Workshop #3 - Javascript Clock

Our in-class Workshop will be to create a clock using only Javascript. The page should display the current time updating every second, and should include a button that when clicked will stop or start the clock.

Please use or demonstrate at least the following javascript techniques in building your clock:

  • Creating Elements
  • Adding Elements to the DOM
  • Using Functions to repeat code
  • Using Timing Tools like 'Set Interval' and 'Set Timeout' to modify the page over time
