Workshop #2 - Responsive Design

Movie Posters

Our in-class Workshop will be to emulate the layout of a movie poster using our CSS Layout skills. You can use any movie poster of your choosing, or feel free to use the Inspiration Links below.

We are not trying to copy the poster exactly, rather just use it as inspiration to practice our layout skills. We are just trying to capture the general layout, rhythm, colors and relationships between the design and text. Try to include at least (3) of the following elements: the Film Title, the Sub-Title, the Credits, and at least one Graphic Element ( this could be a CSS Shape or Images ).

Please use or demonstrate at least the following:

  • Create and import an external CSS file
  • Header Elements ( H1, ... ) for Titles or Sub-Titles
  • List Elements ( UL, OL, LI ) or Paragraph ( P ) elements for additional copy, actor names and credits.
  • Set the Box Model
  • At least one Relatively Positioned element
  • At least one Absolutely Positioned element
  • Use CSS Height and Width to size elements
