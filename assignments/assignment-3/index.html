Assignment #3 - Responsive Logos

Our homework assignment is to create a 'Responsive' Logo. This can be a logo for a real project, or you can feel free to create a new one for a fake project or re-brand an existing one.

The logo should change at least three times - these changes could be at arbitrary points based on your design or you could roughly target (1) mobile, (2) tablet, and (3) laptop screen sizes.Designs must be fully visible and not break on devices of any size or orientation.

Each breakpoint should visually add or remove visual elements, as well as adjusting properties like display, font-size, margins, padding, font-family, and others to effect the overall layout and treatment of different parts of the logo. Try to change at least two properties per screen size.

Take advantage of different kinds of units to cause different effects - "responsive" units like View Height and View Width units can be used to stretch and distort shapes or images ( see The Whitney's Logo ), while unchanging pixel-based units can be used to establish consistent forms.
