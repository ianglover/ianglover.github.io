Assignment #9 - Study Guide for Test

Next week ( May 13 ) we will be taking an in-class test which will be open-note and open-computer. We will be reviewing and testing concepts from the 4 main units, being 1. Interaction with the Internet, 2. Interaction with Type, 3. Interaction with Data, and 4. Interaction with People. From those sections, I would recommend reviewing the following concepts in particular:

Interaction with the Internet

  • Tools ( Git & Github )
    • Know what Git & Github are, and be able to explain the differences between the two.
    • Review Git vs. Github: What's the Difference?
  • Basic HTML & CSS Syntax
    • Understand and be able to explain the role of HTML and CSS in creating webpages, as well as knowing how to use them for basic tasks like creating elements such as paragraphs, lists, headers and applying basic styling to those elements using CSS.
    • Review the Basics of HTML and the Basics of CSS from MDN.
    • Review the HTML/CSS Basics Demo from the Communications Design Vimeo ( password: interaction )
  • Embedded Content
    • Be able to embed content from other places on the internet, such as YouTube, Spotify, or entire other web pages using an iframe.
    • Review the Embedded Content Demo from the Communications Design Vimeo ( password: interaction )

Interaction with Typography

  • Digital Typography
    • Be able to load a custom font locally ( or from Google Fonts or another service ) and be able to use CSS to apply it to your webpage using CSS.
    • Review the Typography on the Web Demo from the Communications Design Vimeo ( password: interaction ).
  • Responsive Web Design
    • Be able to use Media Queries to apply styling to a page based on its screen size using CSS.
    • Review the Responsive Web Styles Demo from the Communications Design Vimeo ( password: interaction ).
  • CSS Layout & Style
    • Review the CSS Layout + Style Demo from the Communications Design Vimeo ( password: interaction ).
  • Multiple Page Websites ( Linking Between Pages )
    • Be able to use Relative or Absolute links to link between different HTML pages that you create.
