Assignment #4 - Album Promotional Website

Our homework assignment is to review everything we have covered in the past six weeks and create a promotional page for an album of your choice.

The design and presentation of the page should emphasize and build upon the visual qualities of the album and its associated content such as music videos, merchandise, or other promotional material.

It should include content from the album such as its artwork, music videos, and promotional material. It should also include information like its tracklisting, record label, release date, credited musicians, and anything else that seems important to include. The website Discogs can be a valuable resource for this kind of information, for example see Caroline Polachek - Pang. You could also include upcoming tour dates, and links to merchandise or album reviews.

The page should use all of the techniques we have learned thus far to demonstrate an understanding of HTML & CSS Basics, Typography, CSS Layout techniques, and Responsive Design.


HTML & CSS Basics

Use the basics of HTML and CSS to lay out your page in a well-organized, legible way using the correct Semantic HTML for each kind of content. For example, use List Elements when listing items and content, and Heading Elements for section headings.

For reference, you can review Assignment 1: HTML/CSS Basics


Files and Folder Structure

Create an organized file and folder structure for the elements of your project, including any external filesheets or local assets like images and custom fonts.


Embedded Content

The page should include:

  • Audio from the album using Spotify, Apple Music, or other online streaming services.
  • Video from the album or artist from YouTube, Vimeo, or other online video services.

    • Typography on the Web

    Include at least one but no more than two custom fonts. Be sure to use the font in multiple sizes, weights, and styles

    For additional reference, please review Assignment 2: Typography on the Web


    CSS Layout

    Use CSS Layout techniques to control the position, size and layout of the page.

    Be sure to demonstrate at least the following CSS Layout techniques:

    • Placing Elements into Rows
    • Placing Elements into Columns
    • Giving Elements Different Widths
    • An Absolute Positioned Element

    For additional reference, please review Assignment 2: Typography on the Web ( Revised )


    Responsive Design

    Use Media Queries and Responsive Design techniques to make the page designs work well on mobile, tablet, and desktop devices.

    For additional reference, please review Workshop 2: Responsive Design.
