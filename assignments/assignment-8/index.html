Core 2: Interaction Lab

Assignment #8 - User Input and Output: User-Created Lists

Our homework assignment is to begin practice using event handlers to allow users to interact with your websites in more complex ways, including using their mouse, keyboard events, and text inputs.

Before starting, I highly recommend watching the two demonstrations I recorded, User Input & Output (1) which is an introduction to event listeners as a concept. There is also User User Input & Output (2) which uses these techniques to create a basic application.

For homework, we will be building a website that lets users create their own lists. What kind of list, and the presentation, is completely up to you! It could be a 'To Do' list, a list of 'Goals' and 'Dreams', a list of favorite bands, arts, or designers. The design of the webpage should emphasize and support the concept of the site, thinking about its layout, typography, and color choices.

The user should be able to use a text input to add items to the list on the page.

There should be at least one other interaction that takes advantage of other events, such as the mouse movement, scrolling, or keyboard events. You could include functional interactions to perhaps remove items from the list, or it could be more experimental and change or update the design .
