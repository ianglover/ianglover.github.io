Assignment #6 - Website as Calendar

Our homework assignment is to create a website which changes over time.

It can change based on a timeline of your choosing - perhaps it changes every minute of the hour and repeats every hour, or changes each day of the year. It could use different timespans as well, changing some elements over the course of the hours of the day and others over the course of the month.

The site can have any purpose, but it should be expressive - it can be purely an abstract generative composition, or it could tell a story that changes over time, but the appearance and functionality of the website should feel like a coherent concept.
