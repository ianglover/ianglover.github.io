Assignment #6 - Databases & APIs

Our homework assignment is to begin learning about how to work with external network resources, such as Databases and APIs elsewhere on the internet. To do so, we will begin by requesting data from an Airtable Database that I have created, and load that data onto our pages.

First, I highly recommend watching these Web Dev Simplified videos. The first is exploring the Fetch API ( 6 Minutes ) which we will be using to make Network Requests.

If you found the concept of 'Promises' introduced in the Fetch API video confusing, then I would recommend also watching the Javascript Promises in 10 Minutes ( 10 Minutes ) video.

Finally, I have recorded a demonstration on API & Databases: Requesting Data and Adding It To Your Page ( 1 hour ), which will pull together all the concepts necessary to complete the homework.

You can also reference and review the Completed Website from the demonstration, as well as its Completed Codebase. I would recommend having the codebase open while watching the demonstration video.
