Github Pages

If you have not already, create an account at Github using your New School email address.

Your account name should be the first part of your New School email address.

For example, if your New School email is gloveri123@newschool.edu then your account should be named gloveri123.


After you have created an account, download the Github Desktop application.

Once downloaded, open the Github Desktop application and Log In using the credentials of the Github account you have just created.



When you have logged in, select File > New Repository.


Give the repository the same Name as your Root Folder, for example glover123.github.io.

In the same window, for Local Path press the Choose... button and find and select your Root Folder



Finally, in the next page you should see an option to Publish Repository, select this option and without changing any options in the window that opens, press Publish Repository again.

If there are no errors, you should now be able to view your Questionnaire online at your Github Page!
